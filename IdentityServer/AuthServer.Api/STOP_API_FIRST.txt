IMPORTANT: STOP THE API FIRST!
======================================

Before rebuilding, you MUST stop Visual Studio's API:

1. In Visual Studio, click the STOP button (red square)
2. Or close Visual Studio completely
3. Or press Ctrl+F5 to stop debugging

Then run:
cd AuthServer.Api
dotnet build
dotnet run
